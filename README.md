# Serprify - Digital Marketing & SEO Agency Website

![Serprify](images/working.png)

## 🎯 About Serprify

**Serprify** is a professional digital marketing and SEO services agency website. The motto is **"Fortify Your Growth"** - helping businesses establish and strengthen their online presence through strategic digital marketing solutions.

## 📋 What is This Project?

Serprify is a fully responsive, modern website for a digital marketing agency that offers:
- **SEO Services** - Search engine optimization to boost online visibility
- **Digital Marketing Consulting** - Strategic guidance for online growth
- **Professional Support** - Dedicated client support services
- **Blog & Resources** - Industry insights and marketing tips

## 🚀 Getting Started

### Prerequisites
- Web browser (Chrome, Firefox, Safari, Edge)
- Local web server or hosting platform

### Local Development
1. Clone the repository:
```bash
git clone https://github.com/msohaiilkz/Serprify.git
cd Serprify
```

2. Open in browser:
   - **Option A**: Double-click `index.html`
   - **Option B**: Use a local server:
     ```bash
     # Python 3
     python -m http.server 8000
     # Then visit: http://localhost:8000
     ```

3. Browse the website:
   - Home: `http://localhost:8000`
   - About: `http://localhost:8000/about.html`
   - Services: `http://localhost:8000/service.html`
   - Blog: `http://localhost:8000/blog.html`
   - Contact: `http://localhost:8000/contact.html`
